# ESP32-LEGO-CAR
<img width="1283" height="604" alt="Screenshot 2026-06-07 055034" src="https://github.com/user-attachments/assets/3ff57224-8db5-4ce1-af7a-a5847c7c94b1" />
So I made a custom ESP32 chip based Car which is based on LEGO theme. I made the PCB on Kicad and the 3D designs were made on autodesk fusion. The board encludes 16 MB flash Storage, USB-C Connector,voltage regulator , battery charging IC , and 2 motor drives. 

## Features

- Base on custom ESP32 chip
- Design Inspired by LEGO
- Includes 4 Motor gears and recharable battery.
